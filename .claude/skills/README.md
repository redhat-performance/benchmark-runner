# Claude Code Skills

Slash commands for common development workflows. Type the command in Claude Code to run it.

| Command | Description |
|---------|-------------|
| `/unittest` | Regenerate golden files and run all unit tests |
| `/integration` | Run integration tests against a live OCP cluster |
| `/push` | Commit all changes, rebase onto main, run unit tests, and push |
| `/review-rabbit` | Review CodeRabbit critical issues on a PR and report real problems |
